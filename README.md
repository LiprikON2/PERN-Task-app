# PERN task app
> Postgres CRUD interface based on PERN stack

### Packages
- express
- cors
- pg
    - Postgres integration
- nodemon
    - Live dev server updates
- nvm-windows
    - Updating node.js

### Running

#### Running server
```bash
nodemon ./server/index.js
```

#### Running client
```bash
npm start --prefix ./client
```
