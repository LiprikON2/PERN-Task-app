-   cors

-   nodemon
    -   Live dev server updates
-   nvm-windows
    -   Updating node.js

### Running

#### Running server

> at localhost:81

```bash
nodemon ./server/index.js
```

#### Running client

```bash
npm start --prefix ./client
```
